# 花卉识别分类系统 python491

## 项目概述

花卉识别分类系统是一款基于深度学习的图像分类应用。该系统使用`Python`语言和`PyTorch`框架，以`resnet18`网络为基础进行构建，旨在实现对十种常见花卉的识别和分类。

## 技术栈

- Python
- PyTorch
- ResNet18

## 功能模块

系统包含以下功能模块：

- **数据预处理**：处理并提供花卉图像数据集。
- **模型训练**：基于提供的花卉数据集，使用`resnet18`网络进行训练。
- **模型评估**：对训练完成的模型进行性能评估。
- **分类识别**：通过训练好的模型对新的花卉图像进行分类识别。

## 系统角色

- **用户**：使用系统进行花卉图像的识别分类。
- **开发者**：负责系统的维护和功能扩展。

## 运行环境

- Python 3.x
- PyTorch 1.x
- 相关依赖库（如：torchvision, numpy, pillow等）

## 部署步骤

1. 确保所有依赖库已正确安装。
2. 下载数据集并进行预处理。
3. 加载预训练权重或从头开始训练模型。
4. 运行系统以执行花卉分类。

## 核心亮点

- **灵活的网络结构**：基于`resnet18`，可以快速迁移到其他分类问题。

## 花卉种类

系统支持以下十种花卉的识别：

- 风铃草
- 金鸡菊
- 金盏花
- 康乃馨
- 玫瑰
- 蒲公英
- 水仙花
- 向日葵
- 郁金香
- 鸢尾花
